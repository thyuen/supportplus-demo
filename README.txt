GitHub Pages demo package

Files:
- index.html
- style.css
- main.js

How to use:
1. Create a GitHub repo
2. Upload all 3 files to the repo root
3. In GitHub: Settings > Pages > Deploy from branch > main > /(root)
4. Open the Pages URL

Notes:
- This is a static demo only.
- Existing page images/CSS dependencies are loaded from public HKU/CDN URLs where possible.
- Search, login, notifications, and other backend features are not expected to work.
- The floating chat widget opens and works as a frontend-only demo.

If layout still looks off on mobile, also upload the original res.css file and update index.html if needed.
